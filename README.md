# Inventory Management
A small store inventory management written in C++
The program asks user to select one option from the menu:
  	
*  0. Exit  
*  1. Item Sale  
*  2. Daily Report 
*  3. Weekly Check
*  4. Monthly Update
*  5. Load Inventory File
*  6. Sort File by Key
*  7. Save Inventory to a File
*  8. Add a new item
*  9. Edit an item
*  10. Delete an item
